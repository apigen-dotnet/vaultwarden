# Apigen.Vaultwarden

Generated C# client for the [Vaultwarden](https://github.com/dani-garcia/vaultwarden) API (Bitwarden-compatible).

This is a unified client generated from 3 OpenAPI specs covering all Vaultwarden API endpoints:

| API | Path prefix | Endpoints |
|-----|-------------|-----------|
| Identity | `/identity` | Register, prelogin, OAuth token |
| Vault | `/api` | Ciphers, folders, sends, organizations, sync, 2FA |
| Public | `/public` | Organization import |

## Installation

```bash
dotnet add package Apigen.Vaultwarden
```

## Usage

Vaultwarden authentication requires PBKDF2/Argon2id key derivation and encrypted vault decryption. See the [example project](examples/Apigen.Vaultwarden.TestClient) for the full login + vault sync + decrypt flow (equivalent to `bw login` + `bw list items`).

## Regeneration

This library is generated by [Apigen.Generator](https://github.com/apigen-dotnet/generator) with multi-spec support.

```bash
git clone https://github.com/apigen-dotnet/generator.git ../generator
dotnet run --project ../generator/src/Apigen.Generator/Apigen.Generator.csproj -- --config specs/vaultwarden.toml
```
